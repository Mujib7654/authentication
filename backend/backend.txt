middleware:
middleware function are function that have access to the request object(req), the response object(res), and the next function in the application request response cycle.

the next function is a function in the express router which, when invocked, executes the middleware succeeding the current middleware.

dotenv: secure your important credentials
installation: npm i dotenv

schema and model:
a document schema is a json object that allows us to define the shape and content of documents and embedded documents in a collection.
It defines the name of each item of data, and the type of data, whether it is string , integer and so on.
